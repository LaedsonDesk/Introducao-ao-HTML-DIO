Como surgiu o HTML

O HTML(Hiper Text Markup Language) foi criada com o objetivo de tornar facilitado o compartilhamento de documentos, com a popularidade da WWW (World Wide Web), tornaria possível o compartilhamento de forma global.

O que é o HTML

Não é uma liguagem de programação, mas sim uma lingugem de marcação, siginifica que ele vai dizer

como que o site vai ser estruturado e quais serão as formas.

O que são TAGs

São formas de estrutura de forma mais específica como que vai ser casa elemento de marcação do documento.

Como itálico, negrito, sublinhado e etc
